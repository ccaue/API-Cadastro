Necessário:

Java 21
Springtools 4
Lombok (arquivo para download está aqui no repositório)
MySQL
Xampp

Passo a passo:

No springtools, crie um novo projeto spring, tipo maven, e vesão 21 do java. Após, selecione as seguintes dependências:

Lombok
MySQL Driver
Spring Boot DevTools
Spring Data JPA
Spring Web

Após a criação do projeto spring, configure o arquivo "aplication.properties" de acordo com a conexão do seu MySQL com o Xampp. 
Depois, basta criar os pacotes e classes e copiar os códigos presentes neste repositório.
